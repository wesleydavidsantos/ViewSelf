## 🔒 Security Model

ViewSelf is designed with a **zero-footprint** security philosophy:

- Connects using **read-only** Oracle credentials
- Creates **no objects** in your database (no tables, no packages, no users)
- Reads only Oracle's built-in system views (`V$SESSION`, `V$SQL`, `DBA_SEGMENTS`, etc.)
- All data stays on your machine — nothing is sent to external servers
- Compatible with environments governed by strict security policies
